My responsibility

프로젝트를 성공적으로 완성하기 위해서
아래와 같은 업무를 진행해왔습니다.

  • 수행 기간: 2023.11.15 ~ 2023.12.27 (1개월)
  • 참여 인원: 5명
  • 기여도: PE/BE: 65%,Data-Base:25%,Design: 10%

공공데이터 포털 및 웹사이트에서 대기오염 성분에 따른 질병 내용 데이터 수집 및 CSV화 하여, 대기오염 정의와, 인체에 미치는 질병을 포토샵을 이용하여 시각화하여 심각성 부여, Chart.js와 BootSrapt을 이용하여 버튼클릭 시, 질병연관도 그래프를 표현하였습니다.

OPEN API인 기상청과 한국환경공단 API를 활용하여, 대기오염 농도에 따른 색이 변화되는 카토그램을 구현하여 현재 대기상태를 보여주며, 기상청 정보를 함께 제공하고 있습니다 추가로 케이웨더 홈페이지에서 생황지수 부분을 Crawling하여 지수 그래프와 멘트를 표현하였습니다..

캘린더의 날짜를 클릭 시, 모달 창이 뜨면서, 출발지와 도착지를 입력하면 이동거리 계산을 자동차 탄소배출량 기준이며, 산출식은 탄소발자국 사이트를 참고하여, Google API와 Kakao API를 통해 탄소배출 감소량을 계산해줍니다. 흡연자의 경우 담배 개비 수에 따른 탄소 배출량을 질병관리청 (주간 건강과 질병•제15권 제22호) 를 참고하여 계산해 하루의 총 예상 감소량을 안내해주는 계산기 기능입니다. 산출된 감소량을 스탬프와 함께 캘린더에 기록하며, 기록된 데이터는 SQL Query문, REST API, JDBC와 Ajax을 통해 Oracle DB에 저장하여 관리합니다

Asside를 통해캘린더에 저장된 DB 데이터를 SQL Query문, REST API와 Ajax을 이용하여 모든 사용자의 캠페인 참여도와 로그인 된 사용자의 캠페인 참여도를 표기하여, 캠페인 참여에 동기부여 Admin Page를 통해 관리자 페이지 구축하여, 모든 사용자의 리스트 구현하여 사용자 관리(삭제/활동내역) Chart.js를 통해 모든사용자와 선택한 사용자의 활동 내역 시각화

CSS, HTML, JavaScript, Bootstrap 이용 사용자가 손쉽게 이용할 수 있도록 화면을 구상하였습니다.

Goals and Lessons

위 프로젝트로 대기오염의 심각성을 시각적으로 표현하여, 경각심을 부여하고 미세미세의 서비스에 추가적인 서비스로 실시간 대기상황과 기상정보, 생활지수를 확인하고 탄소절감 캠페인에 동참하여 친환경활동에 참여할 수 있도록 하는 것을 목표로 제작하였습니다.
